# XAI_Medical
Built an Explainable AI (XAI) medical diagnosis system using EfficientNet-B0 for X-ray analysis, achieving 91.3% accuracy (AUC 0.95) for pneumonia detection and 88.7% accuracy (AUC 0.92) for bone fracture detection. Integrated preprocessing, Grad-CAM++ heatmaps, and confidence scores for interpretable predictions.
